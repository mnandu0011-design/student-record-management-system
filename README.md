# 🎓 Student Record Management System using Singly Linked List

A Python-based **Student Record Management System** built using the **Singly Linked List** data structure. This project demonstrates the practical implementation of linked list operations without using Python's built-in list for data storage. It is designed to strengthen **Data Structures and Algorithms (DSA)** concepts through a real-world application.

---

## 📌 Features

- ➕ Add a new student record
- 📋 Display all student records
- 🔍 Search for a student by Roll Number
- ✏️ Update student information
- ❌ Delete a student record
- 📊 Count total student records
- 🖥️ Menu-driven command-line interface

---
